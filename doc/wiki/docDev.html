RedNetwork : Le meilleur opensource

Voici une documentation qui facilitera l'utilisation du code pour les développeurs sachant qu'il faut au préalable être dans le dossier src :

Pour compiler le serveur, le client et le filtre :

Concernant le serveur, il FAUT, pour la phase de compilation être root. En suite, taper make.

Pour exécuter le serveur:

Toujours en étant root bien sûr, faire appel à l'exécutable "./server".

Pour exécuter le client:

Plusieurs possibilités se présentent :
  • ./client hostname get distfilename localfilename
  • ./client hostname list
  • ./client hostname sniff protocole_name

    • Pour exécuter le filtre:

    Il y a possibilité d'exécuter le filtre de différentes manières :
  • ./filter file.pcap num_packets /* the number of packets to sniff */
  • ./filter file.pcap /* by default it sniffs all the packets */
